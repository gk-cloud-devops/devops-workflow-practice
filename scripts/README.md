# Bash Automation Scripts

This directory contains basic automation scripts used in Linux system administration.

Scripts included:


disk_alert.sh – Checks disk usage and alerts if threshold exceeds  
system_health.sh – Displays CPU, memory, and disk usage  
log_checker.sh – Removes old log files
service_monitor.sh - It monitor the service status, if service is not running or active it restart the service automatically. 
website_health.sh - It check the website status, if website is not responding it restart's.

These scripts were created as part of DevOps learning and automation practice.
